📊 评估层
🧮 Evaluator
📊 提供分类任务的评估方法
📝 文本生成任务的评估（如 BLEU, Rouge）
📊 JSON 结构和字段检查
📈 扩展性：支持自定义评估标准和指标
评估层负责对模型输出进行各种评估，支持分类和生成任务。
📊 常见指标
🎯 准确率（Accuracy）
🎯 精确率（Precision）
🎯 召回率（Recall）
🎯 F1 分数
使用常见的分类任务指标评估模型性能。
📊 数据处理层
💻 DataProcessor(根据Task不同进行不同的处理)
📑 支持从不同数据结构（如 DataFrame、CSV）中提取数据
✏️ 可扩展性：支持数据预处理（如文本清理、规范化）
📦 处理结果可以动态存储到不同数据格式（如新增列或导出文件）
数据处理层负责从不同数据源中提取和处理文本数据。
⚙️ 自定义操作
✂️ 支持用户自定义列操作
🔍 自动提取和处理文本数据
用户可以根据需求自定义数据处理步骤。
🧠 模型层
🎯 ModelAPI
🔗 封装模型 API 调用
⚙️ 动态配置（如模型名称、API 版本）
🛠 统一接口：`analyze_text`
模型层负责通过API与不同的NLP模型进行通信。
