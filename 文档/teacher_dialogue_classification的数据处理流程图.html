数据处理和模型调用流程图

输入参数
model_parameters, data, data_processor
参数校验
检查 data_processor, data 是否包含必须的键，且 data 中的键值为列表类型。
数据处理
使用 DataProcessor 对 data 进行处理。
参数：
task: data_processor['Task']
T: data_processor['T']
模型调用
如果存在 model_parameters，则调用模型。
参数：
model_family, api_key, model_name, api_version, prompt
输出结果
输出经过模型分析的结果，包含处理过的 teacher_text 和 student_text。
